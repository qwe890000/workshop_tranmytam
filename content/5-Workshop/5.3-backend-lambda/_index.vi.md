# Backend Lambda

## Tổng quan

Phần này hướng dẫn chúng em tạo Backend Serverless sử dụng AWS Lambda và API Gateway để xử lý logic tính toán thần số học.

## Nội dung

1. **5.3.1 - Tạo Lambda Function** - Viết Lambda để tính toán các chỉ số thần số học
2. **5.3.2 - Configure API Gateway** - Tạo REST API endpoint để gọi Lambda

## Thứ tự thực hiện

```
5.3.1 Create Lambda → 5.3.2 Configure API Gateway
```

## Các bước tiếp theo

Bắt đầu với phần **Tạo Lambda Function** để triển khai backend tính toán thần số học.
