# Triển khai Frontend

## Tổng quan

Phần này hướng dẫn chúng em triển khai giao diện Frontend của ứng dụng LunaGenZ lên AWS.

## Nội dung

1. **5.5.1 - S3 Hosting** - Upload static files lên S3 và bật static website hosting
2. **5.5.2 - CloudFront CDN** - Phân phối nội dung toàn cầu với HTTPS

## Thứ tự thực hiện

```
S3 Hosting → CloudFront CDN
```

## Các bước tiếp theo

Bắt đầu với phần **S3 Hosting** để upload frontend lên AWS.
