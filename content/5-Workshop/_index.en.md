# Deploy LunaGenZ (Numerology) on AWS Serverless

#### Overview

**LunaGenZ** is a Numerology application built entirely on AWS Serverless architecture. This workshop will guide you through the entire process of deploying Backend logic, automating internal PDF report generation, and hosting the Frontend application.

In this lab, we have:
+ Built and deployed Serverless Backend using AWS Lambda and API Gateway.
+ Automated the process of generating PDF reports based on Numerology metrics.
+ Hosted the Frontend interface ensuring high availability on Amazon S3 and Amazon CloudFront.
